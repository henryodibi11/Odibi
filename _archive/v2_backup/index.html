    🎯 Common Tasks

    How do I...?

    Run a pipeline from YAML?

    from odibi.pipeline import PipelineManager
manager = PipelineManager.from_yaml("config.yaml")
results = manager.run()

    See: Quick Start Guide

    Add a new connection?

    connections:
  my_data:
    type: local
    base_path: /path/to/data

    See: Configuration Guide

    Transform data with SQL?

    nodes:
  - name: clean_data
    transform:
      steps:
        - "SELECT * FROM previous_node WHERE amount > 0"

    See: User Guide

    Use Delta Lake?

    write:
  format: delta
  mode: append
  options:
    versionAsOf: 5  # Time travel!

    See: Delta Lake Guide

    🆘 Troubleshooting

    See the complete Troubleshooting Guide for: - Configuration errors - Execution issues - Common pitfalls - Debugging strategies

    Quick fixes: - "ValidationError: field required" → Check Configuration Guide - "Pipeline 'X' not found" → Verify pipeline names in YAML - "Connection 'X' not configured" → Add to connections: section - SQL errors → Table names must match node names
