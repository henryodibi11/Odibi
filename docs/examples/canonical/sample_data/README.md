# Sample Data for Canonical Examples

Copy these files to your project's `data/landing/` folder to run the examples.

| File | Used By |
|------|---------|
| `customers.csv` | Examples 1, 3 (Hello World, SCD2) |
| `orders.csv` | Examples 4, 5 (Fact Table, Full Pipeline) |
| `products.csv` | Example 4 (Fact Table) |

## Setup

```bash
# From your project root
mkdir -p data/landing
cp docs/examples/canonical/sample_data/*.csv data/landing/
```
