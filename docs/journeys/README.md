# Learning Journeys

> **"We Win When You Win"** — Choose your path and start learning.

## Choose Your Journey

Each journey is designed to take you from zero to productive in the shortest time possible. Pick the one that matches your role and goals.

| Role | Journey | Time | What You'll Build |
|------|---------|------|-------------------|
| 📊 **Business Analyst** | [BA Journey](business-analyst.md) | 2-3 hours | Read Data Stories, interpret DAGs, validate quality |
| 🎓 **Junior Data Engineer** | [Jr DE Journey](junior-data-engineer.md) | 4-6 hours | Run pipelines, copy configs, debug errors |
| 🚀 **Senior Data Engineer** | [Sr DE Journey](senior-data-engineer.md) | 1-2 days | Design layers, productionize, custom transforms |
| ⚗️ **Chemical Engineer / SME** | [ChemE Journey](chemical-engineer.md) | 2-3 hours | Frame sensor/batch data, validate business rules |

---

## What Makes These Journeys Special?

### 1. **No Prerequisites Assumed**
Every journey starts from zero. If you can open a terminal, you can complete the journey.

### 2. **Hands-On Verification**
Every module includes "Do" and "Verify" steps. You'll know immediately if you got it right.

### 3. **Clear Outcomes**
Each journey ends with a capstone project that proves you can do real work.

### 4. **Prescriptive Next Steps**
We tell you exactly where to go after completing your journey.

---

## Learning Path Structure

Each journey follows this template:

```
📌 Who Is This For?
⏱️ Time to Complete
🎯 Learning Outcomes
📋 Prerequisites
```

Then, for each module:

```
📖 Read (concept explanation)
👀 Watch (optional video)
✋ Do (hands-on exercise)
✅ Verify (check your work)
```

Finally:

```
🏆 Capstone (prove you can do it)
➡️ Next Steps (where to go from here)
```
